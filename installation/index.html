    Installation

    Requirements

    • NeoVim 11.0+ (older versions will not work properly)
    • git
    • Optional (Windows): Zig (choco install zig) or GCC via Scoop

    Manual Installation

    1. Backup your current config:

    # macOS/Linux
mv ~/.config/nvim ~/.config/nvim.bak

# Windows PowerShell
Rename-Item -Path "$env:LOCALAPPDATA\nvim" -NewName "nvim.bak"

# Windows CMD
cd %LOCALAPPDATA%
ren nvim nvim.bak

    2. Clone ZaeNvim:

    # macOS/Linux
git clone https://github.com/zaetrix/zaenvim.git ~/.config/nvim/

# Windows
git clone https://github.com/zaetrix/zaenvim.git %LOCALAPPDATA%/nvim

    3. Optional: Remove GitHub repo artifacts

    # macOS/Linux
cd ~/.config/nvim
rm -rf .git README.md LICENSE

# Windows PowerShell
cd $env:LOCALAPPDATA\nvim
Remove-Item .git, README.md, LICENSE -Force -Recurse

# Windows CMD
cd %LOCALAPPDATA%\nvim
rmdir /s /q .git

    4. Launch NeoVim

    nvim

    Let the plugins install automatically on first run.
